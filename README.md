# Masakala Senior Secondary School Website

This is the official website for Masakala Senior Secondary School, a public no-fee secondary school located in Masakala A/A, Matatiele, Eastern Cape.

**Motto:** "Reach for the Sky"

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
